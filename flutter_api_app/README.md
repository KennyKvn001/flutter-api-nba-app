# NBA App - API

This project is a Flutter application that uses the BallDontLie API to fetch and display NBA teams, players, and games. It also utilizes the Provider state management for efficient data handling.

api referrence: https://docs.balldontlie.io/#nba-api

## Features

- **NBA Teams:** View a list of all NBA teams, along with their details and statistics.
- **NBA Players:** Explore the different of NBA players, including their position and current teams.
- **NBA Games:** Stay updated with the latest NBA games, including live scores and game details.

## Navigation

- **Home Screen:** The home screen displays a list of NBA teams. Tap on a team to view its details.
- **Team List:** Here, you can see the team list.
- **Player List:** Here, you can see the players list.
- **Game List:** Here, yuo can see the game list.
